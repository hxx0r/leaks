# leaks
Приложение для проверки наличия пароля в слитых базах данных

Технологии:
Kotlin - основной язык программирования 
XML - разметка пользовательского интерфейса 
OkHttp - сетевые запросы Эффективная работа с HTTP, поддержка корутин. (Coroutines) 
Асинхронное программирование - упрощение работы с потоками
Material Design - дизайн-система 
SHA-1 - Криптографическое хеширование, стандарт для API Have I Been Pwned
MVVM Архитектура приложения -разделение логики и интерфейса, тестируемость

1. Пароль не отправляется в интернет
2. На устройстве вычисляется SHA-1 хеш пароля
3. Отправляется только первые 5 символов хеша (префикс)
4. Сервер возвращает список хешей с таким префиксом
5. Сравнение происходит локально на устройстве
6. Метод k-анонимности

Изучаемые темы:

1. Мобильная разработка на Kotlin/XML
2. Работа с сетью и REST API
3. Криптография и хеширование
4. Безопасность данных и приватность
5. Асинхронное программирование (корутины)
6. Архитектура приложений (MVVM)
7. Material Design и UI/UX дизайн
8. Тестирование и отладка
